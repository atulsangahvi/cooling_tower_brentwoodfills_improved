# Enhanced Cooling Tower Design Tool

A comprehensive Streamlit application for cooling tower thermal design with realistic fill data and Merkel method verification.

## Features

- **Realistic Fill Database**: Based on actual manufacturer specifications (Brentwood, Marley, EVAPCO)
- **Merkel Method Verification**: Checks if selected fill can meet thermal requirements
- **L/G Ratio Optimization**: Finds optimal water-to-air mass flow ratio
- **Enhanced Pressure Drop**: Includes water loading effects and realistic exponents
- **Safety Factors**: Configurable thermal and design margins
- **PDF Reporting**: Generates comprehensive design reports
